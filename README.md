# Nuxt Starter PWA

> A nuxt starter project that does all the boring parts for you.

## Features

-   Sass installed
-   Prettier, ESLint & Stylelint
-   Nuxt modules installed by default:
    -   @nuxtjs/pwa
    -   @nuxtjs/dotenv
    -   @nuxtjs/component-cache
-   Webpack plugins
    -   Imagemin
-   Vue-lazyload installed by default.
